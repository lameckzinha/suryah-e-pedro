Tecnologias

HTML (Hypertext Markup Language):HTML é a linguagem de marcação padrão para a criação de páginas web. Utiliza tags para estruturar o conteúdo, como texto, imagens e links, permitindo a criação de páginas interativas.

CSS (Cascading Style Sheets): CSS é uma linguagem de estilo que complementa o HTML, controlando a aparência e o layout dos elementos na página. Define propriedades como cor, fonte e posicionamento.

JavaScript:JavaScript é uma linguagem de programação de alto nível utilizada para criar interatividade nas páginas web. Pode ser executado no navegador do usuário, manipulando dinamicamente o conteúdo da página.
